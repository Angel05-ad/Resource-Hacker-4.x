   Click on *File > Open*. Select any `.exe` or `.dll` file on your computer. For example, try a program in your `C:\Program Files` folder.

   Resources are grouped by type. Click any folder like *Icon* or *Dialog* to see contents.

   Select a resource and press *Action > Replace Resource* or *Action > Add a New Resource* to insert your own files.

5. **Save your work**

## 🔧 Tips for Safe Editing

- Always back up original files before modifying.  
- Work on copies, not the original files on your system.  
- Some system files may require administrator rights to edit.  
- Incorrect changes can cause programs to stop working. Only change files if you understand the action.  
- Do not edit files that update automatically, or you might lose changes after an update.

## ❓ Troubleshooting

- **Program won’t open:** Make sure you have downloaded the correct `.exe` file and that your Windows version supports it.  
- **Cannot save changes:** Check if the file is read-only or if you have permission to edit it. Run Resource Hacker as Administrator for system files.  
- **File becomes corrupted:** Always save to a new file. Restore your original file if problems happen.  
- **Unable to replace resource:** Some resources are locked by the program or Windows and cannot be edited.
